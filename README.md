# Acute Oral Toxicity Prediction

A Streamlit-based web application for predicting **acute oral toxicity** of chemical compounds from **SMILES strings**.

## Features
- Draw molecule using an interactive molecular editor
- Predict acute oral toxicity from SMILES
- Display toxicity probability
- Batch prediction from CSV
- Chemical structure visualization

## Files
The application uses:
- `toxicity_streamlit_app.py`
- `Stack_Top5.joblib`
- `final_selected_features.joblib`
- `median_imputer.joblib`
